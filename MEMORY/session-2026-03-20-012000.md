# Session Log: AgentTrack Product Design + Implementation

**Date**: 2026-03-20 01:20

## Summary
Designed and implemented AgentTrack — an open-source Rust TUI + Web dashboard for monitoring Claude Code agent teams in real-time. Went from idea to working MVP in one session.

## Key Decisions
1. **Product name**: AgentTrack
2. **Target user**: Individual developers using Claude Code agent teams
3. **Tech stack**: Rust + Ratatui (TUI) + axum (Web) — chose Rust over Go because AI coding neutralizes dev speed disadvantage, and Rust gives smaller binary (2.4MB), faster startup, lower memory
4. **Data sources**: File watching (~/.claude/ JSON files) + Claude Code hooks (PostToolUse)
5. **UI**: TUI primary (k9s-style 4-panel), Web UI optional (SSE dark theme dashboard)
6. **Web UI v0.2 planned**: D3.js network topology, Matrix-style code rain, share screenshots
