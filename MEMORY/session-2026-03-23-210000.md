## Summary
Major cctrack review + refactor session. Renamed agenttrack → cctrack, comprehensive UX review from user perspective, Web UI overhaul, session management improvements, and multiple rounds of color theme iteration.

## Key Decisions
- **Folder renamed**: `agenttrack/` → `cctrack/`
- **Sessions panel**: removed MODEL column on ALL tab, status dot inline with name (● Name)
- **Session naming**: "Project: title" format (e.g. "Clipal: relect 一次整个项目"), project name highlighted in blue
- **Session title**: read from transcript's first meaningful `queue-operation` content, retry until found, 32 char limit
- **Startup scan**: scan last 24h transcripts to pre-populate sessions on restart
- **Session timeout**: transcript file mtime based (5min→Idle, 30min→Shutdown), more accurate than hook-based
- **Team expiry**: all agents shutdown + 60min no activity → hide from tab bar
- **Sub-agents**: no longer filtered, show in ALL tab alongside parent sessions
- **Activity order**: newest at bottom (tail style, developer intuition)
- **Theme**: ANSI native colors (terminal decides actual colors) + Ghostty `#292c33` bg
- **ALL tab layout**: Sessions 55% top + Stats | Activity fills remaining (no Messages panel)
- **30s Tick event**: periodic status refresh without waiting for new hooks
